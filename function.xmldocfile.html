xmldocfile

(PHP4 >= 4.0b4)

xmldocfile — Creat un objecto DOM a partir de un archivo XML

Descripción

object xmldocfile (string filename)

Esta función analiza el documento XML contenido en el archivo referido en filename y devuelve un objecto de clase "Documento Dom", que tiene las propiedades "doc" (resource), "version" (string) y "typo" (long).
