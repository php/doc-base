Que se puede hacer con PHP?

Al nivel mas basico, PHP puede hacer cualquier cosa que se pueda hacer con un script CGI, como procesar la informacion de formularios, generar paginas con contenidos dinamicos, o mandar y recibir cookies.

Quizas la caracteristica mas potente y destacable de PHP es su soporte para una gran cantidad de bases de datos. Escribir un interfaz via web para una base de datos es una tarea simple con PHP. Las siguientes bases de datos estan soportadas:

Adabas DInterBaseSolid
dBasemSQLSybase
EmpressMySQLVelocis
FileProOracleUnix dbm
InformixPostgreSQL 

PHP tambien soporta el uso de otros servicios que usen protocolos como IMAP, SNMP, NNTP, POP3, e incluso HTTP. Tambien se pueden abrir sockets de red directos (raw sockets) e interactuar con otros protocolos.
