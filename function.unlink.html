unlink

(PHP3 , PHP4 )

unlink — Borra un fichero

Descripción

int unlink (string filename)

Borra el fichero filename. Es similar a la función unlink() del Unix C.

Devuelve 0 o FALSE en caso de error.

Ver también rmdir() para borrar directorios.
