method_exists

(PHP4 >= 4.0b2)

method_exists — Comprueba que el metódo de clase existe

Descripción

bool method_exists (object object, string method_name)

Esta función devuelve verdadero si el metódo referido por method_name ha sido definido en el objecto object, en cualquier otro case devuelve falso
