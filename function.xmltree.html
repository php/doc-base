xmltree

(PHP4 >= 4.0b4)

xmltree — Crea un árbol de objectos PHP a partir del un document XML

Descripción

object xmltree (string str)

Esta función analiza el documento XML contenido en el texto str y devuelve in árbol de objectos PHP representando el documento analizado.
